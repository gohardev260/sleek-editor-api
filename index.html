Integration Blueprint

1. Installation

Include the CSS stylesheet and JS library files locally or from your server host.

<!-- Load standalone stylesheet -->
<link rel="stylesheet" href="editor.css">

<!-- Load core class -->
<script src="editor.js"></script>

2. Initialization

Mount a new instance to a container element and define callback hooks.

const editor = new DiginixEditor({
    container: '#editor-mount',
    initialHTML: '<p>Start text...</p>',
    onChange: (html) => {
        console.log("Content changed!", html);
    }
});
